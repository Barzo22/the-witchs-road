# The Witch's Road

A wave-based survival game developed in Unity as a team project.
Survive as long as possible while upgrading your attacks and defeating enemies.

## My Contribution
- Experience system and level progression
- Attack upgrade system
- Experience orb system (drop, attraction and collection)
- Enemy system bug fixes and improvements
- Architecture and design patterns implementation

## Design Patterns Used
- **Flyweight** — efficient enemy data management
- **Factory + Object Pool** — enemy and orb spawning and reuse
- **Observer (EventManager)** — decoupled event communication between systems

## Gameplay Features
- Two attack types: sustained left-click attack and area projectile
- Wave-based enemy spawning
- Experience orbs dropped by enemies
- Attack upgrade system on level up

## Built With
- Unity 2022.3
- C#
- SOLID principles
- Design patterns
